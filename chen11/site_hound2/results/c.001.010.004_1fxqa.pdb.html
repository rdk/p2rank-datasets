Summary of predicted binding sites ranked by Total Interaction Energy. Each cluster can correspond to a binding site or a subset of a binding site.
Cluster Number: rank order of the cluster (click on it to display additional information in "Cluster Details" window below).
Total Energy: Sum of the interaction energy of all points contributing to this cluster. It is a measure of the "strength" of the binding site.
Energy Range: Lowest and highest interaction energy for all points contributing to this cluster. It is possible to color cluster points according to the associated interaction energy using the "Cluster Color" pull-down menu in the "Cluster Display" section.
Cluster Center: coordinates of the Center of Energy (average of the coordinates of the points that belong to the cluster, weighted by interaction energy)
Cluster Volume: Size of the cluster in cubic A.
Protein residues within 4.0 angstroms of the selected cluster. Data shown is: Cluster number, Amino acid, Residue number (in PDB file), and Protein chain.
Cluster Data: plain text file containing data for all clusters (not just the first 10).
Cluster DX file: File in APSB-style format for displaying clusters in programs such as PyMOL and Chimera.
MAP file: File required for SiteHound standalone program.
Cluster PDB file: File in PDB format for visualizing clusters. Each cluster is one "residue" labeled according to the cluster rark (e.g. the first cluster is residue "C01", the second "C02", etc.)
Protein PDB file: Processed PDB file used by SiteHound-web.
